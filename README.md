# Valentine-Page-Maker
Valentine Page Maker - where users pick a template, upload photos, add a message + music link, preview, then pay to publish a shareable link + QR code.

## Environment variables

Set these for local dev and deployment:

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY` (server-only)
- `NEXT_PUBLIC_SITE_URL`
- `STRIPE_SECRET_KEY`
- `STRIPE_WEBHOOK_SECRET`
- `STRIPE_PRICE_NORMAL`
- `STRIPE_PRICE_PRO`
